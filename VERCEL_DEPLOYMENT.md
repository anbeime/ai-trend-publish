# 🚀 Vercel 部署指南（推荐）

## ✅ 优势
- 免费部署
- 全球 CDN
- 自动 HTTPS
- 无需命令行

## 📋 部署步骤

### 方法一：直接拖拽部署（最简单）
1. 访问 https://vercel.com
2. 用 GitHub 账户登录（推荐）或邮箱注册
3. 点击 "New Project"
4. 直接拖拽 `c:/D/ai-trend-publish` 文件夹到页面
5. 点击 "Deploy"

### 方法二：Git 部署
1. 将代码上传到 GitHub
2. 在 Vercel 中导入项目
3. 自动部署
