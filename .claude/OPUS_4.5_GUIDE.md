## 🔄 如何切换到 Opus 4.5

### 方法1: 使用 /model 命令（推荐）

在 Claude Code 对话中输入：

```
/model
```

然后选择：

### 方法2: 在新对话开始时指定

启动 Claude Code 时：

```bash
claude --model opus-4.5
```

或者在配置中设置默认模型。

## 💡 使用建议

### 何时使用 Opus 4.5

✅ **推荐场景**：
- 🧠 复杂的逻辑推理和数学问题
- 📊 深度数据分析和报告生成

## ⚙️ 配置默认模型

### 临时切换

```bash
# 在对话中
/model opus-4.5
```

### 持久配置

创建或编辑 `~/.claude/config.json`：

```json
{
  "defaultModel": "claude-opus-4-5",
  "models": {
    "opus": "claude-opus-4-5",
    "sonnet": "claude-sonnet-4-5",
    "haiku": "claude-haiku-3-5"
  }
}
```

### 项目级配置

在项目目录创建 `.claude/config.json`：

```json
{
  "defaultModel": "claude-opus-4-5",
  "settings": {
    "temperature": 0.7,
    "maxTokens": 4096
  }
}
```

## 📝 常见问题

### Q: Opus 4.5 和 Sonnet 4.5 差别大吗？

A: 对于**日常编程任务**，Sonnet 4.5 完全够用。Opus 4.5 在**复杂推理、架构设计、深度分析**方面有明显优势。

### Q: 如何知道当前用的是什么模型？

A: 在对话中使用 `/model` 命令查看，或直接询问 "你是什么模型？"

### Q: 可以中途切换模型吗？

A: 可以！使用 `/model` 命令随时切换。

### Q: Opus 4.5 比 Sonnet 4.5 慢多少？

A: 大约慢 1-2 倍，但质量提升明显。

### Q: 有 token 限制吗？

A: 不同模型有不同的上下文窗口，但都支持较长对话。
